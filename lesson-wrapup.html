Elliptic Curve is a very popular method for encrpting data. There are many standard values used in web browsers and other programs to handle setting up an encryted session for you, such as when using an ecrypted URL and the 'https://' tag. The National Institude of Standards and Technology (NIST) publishes a list of suggestions every few years. The latest of which, NIST FIPS 186-4, includes widely accepted standards such as P-192, P224 and P-256. In comparison to the values we used for this lesson, Bitcoin uses the the values below, also known as curve secp256k1. While a and b are still small, the value for Z is quite large, as is the generator value G.
